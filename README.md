# Dokumenty.TV (CZ/SK) Stremio Addon - Release 2

**Stav:** Stabilní (Release 2)  
**Verze:** 2.0.0

Tento projekt poskytuje **neoficiální** Stremio addon pro přístup ke katalogu dokumentárních filmů ze stránek Dokumenty.TV. Addon používá web scraping k nalezení veřejně vložených videí (embedů z YouTube, Vimeo apod.).

## Hlavní vlastnosti

- Katalog: Procházení dokumentů v kategoriích (Poslední přidané, Nejsledovanější, Historie, Příroda, Technika, Krimi).
- Streamování: Získání přímého odkazu na video pro přehrání ve Stremiu.
- Nasazení: Vhodné pro bezplatné hostování (např. Render).

## Struktura repozitáře

- `package.json` — závislosti a skripty.
- `manifest.json` — metadata pro Stremio.
- `addon.js` — hlavní server a scraping.
- `README.md`, `INSTALACE.md`, `O_PROJEKTU.md` — dokumentace.

Viz také `INSTALACE.md` pro návod na nasazení.
